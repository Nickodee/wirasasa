# WiraSasa - Complete Screen List

## Authentication Screens
1. **LoginScreen** - User login with email/password
2. **RoleSelectionScreen** - Choose between Client or Provider role

## Client App Screens

### Main Screens
1. **HomeScreen** - Service categories, search, and "How it works" section
2. **ServiceSelectionScreen** - Map view with available professionals for selected service
3. **ProviderProfileScreen** - View provider details, ratings, pricing, and request service
4. **ChatScreen** - Real-time messaging with service provider
5. **TrackingScreen** - Live map tracking with ETA and communication options

### Additional Screens
6. **JobHistoryScreen** - View all past and current service requests
7. **ReviewScreen** - Leave reviews and ratings for completed services
8. **ProfileScreen** - User profile, settings, and account management

## Provider App Screens

### Main Screens
1. **DashboardScreen** - Weekly earnings chart, total earnings, online/offline toggle
2. **JobInProgressScreen** - View and manage active job details, navigate to location

### Additional Screens
3. **JobHistoryScreen** - View all completed jobs with earnings summary
4. **ProfileScreen** - Provider profile, settings, payment methods, job history

## Navigation Structure

### Client Navigation
- Home → Service Selection → Provider Profile → Chat/Tracking
- Home → Profile → Job History → Review
- All screens accessible via navigation stack

### Provider Navigation
- Dashboard → Job In Progress
- Dashboard → Profile → Job History
- All screens accessible via navigation stack
